🧠 AI/ML Curriculum — Mohamed Badawi

A structured, hands-on curriculum covering the full AI/ML stack — from mathematical foundations to production-grade systems. 

Built from real engineering experience, not just textbooks.


👋 About This Repo
This is the course material I developed and teach as an AI/ML Instructor — covering everything a practitioner needs to go from zero to building and deploying real AI systems.
The curriculum reflects how I actually work: grounded in math, built on clean code, and always pointed toward production.

📚 Curriculum Overview
1. 🐍 Python for AI

Python fundamentals for data and ML workflows
Data structures, comprehensions, and functional patterns
Working with NumPy and Pandas efficiently

2. 📊 Exploratory Data Analysis (EDA)

Understanding data distributions and relationships
Visualization with Matplotlib and Seaborn
Detecting outliers, missing data, and data quality issues

3. ⚙️ Data Preprocessing Pipelines

Feature encoding, normalization, and scaling
Handling imbalanced datasets
Building reusable, production-ready preprocessing pipelines

4. 🧮 ML & DL Mathematical Foundations

Linear algebra, calculus, and probability for ML practitioners
Gradient descent and optimization from first principles
Bias-variance tradeoff, regularization (L1/L2), and generalization
Neural network forward/backward pass — derived, not memorized

5. 🤖 Machine Learning Algorithms (7+ covered)
Each algorithm is covered with:

Intuition and mathematical derivation
Implementation from scratch
Scikit-learn usage and best practices
When to use it — and when not to

Algorithms covered include:
Linear Regression · Logistic Regression · Decision Trees · Random Forests · K-Nearest Neighbors · Gradient Boosting

6. 🧬 Deep Learning & Neural Networks

Feedforward networks, activations, and loss functions
Convolutional Neural Networks (CNNs)
Recurrent architectures and sequence modeling
Training dynamics: learning rate schedules, dropout, batch norm

7. 📏 Evaluation & Model Selection

Classification and regression metrics — chosen for the problem, not convention
Cross-validation strategies
Hyperparameter tuning: grid search, random search, Bayesian optimization
Feature importance and model interpretability
